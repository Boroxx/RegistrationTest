Aktive Aufträge

  • Aktive Aufträge sind Angebote,die von Unseren Kunden akzeptiert worden sind. Aktive Aufträge können desweiteren an Unsere Nachunternehmer verteilt werden.

  • Wird ein Auftrag einem Nachunternehmer vergeben, so kommt dieser in die Phase "Vergebener Auftrag".

  • Ein Vergebener Auftrag wird dann vom Nachunternehmer bearbeitet

  • Ist dieser Auftrag fertiggestellt so wird der Auftrag zu einem abgeschlossenen Auftrag.
